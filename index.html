👋 Welcome to DataProxy Studio

This tool documents and explores the CVS Health Data Proxy System — the machine learning architecture that let CVS "see around the COVID corner" by replacing unreliable historical sales data with five alternative proxy signals during the pandemic. Use the interactive diagrams to understand the data design, explore forecasts, and monitor proxy source health.

📐 Conceptual Data Model (CDM)

The CDM is the big-picture business view. It shows what data the business needs — entities and their relationships — without any technical detail. Designed for stakeholders, not engineers.

  • Drag any entity to rearrange the layout
  • Scroll to zoom in/out on the canvas
  • Double-click an entity to edit its name and description
  • Click ⟶ Relationship to connect two entities
  • Click ＋ Entity to add a new business concept
  • Right-click an entity for more options
  • Del key deletes the selected entity
  • Use ⊡ Fit to auto-fit all entities on screen

🗄️ Logical Data Model (LDM)

The LDM is the technical data designer view. It shows the same entities but with attributes, data types, and key constraints — the structure a developer or data engineer would work from.

  • Same drag/zoom/edit controls as the CDM
  • Each entity box shows: 🔑 PK (primary key), 🔗 FK (foreign key), ◆ AK (alternate key)
  • Attribute data types are shown on the right of each row
  • Double-click to edit entity name or description
  • Changes auto-save to browser storage — refresh-safe
💡 Tip: Use ⟳ Reset to restore the original default layout at any time.

📖 About the CVS Health Data Proxy Case

During COVID-19 (2020–21), CVS Health faced a critical challenge: historical sales data — normally the foundation of demand forecasting — became unreliable due to panic-buying, stockouts, and rapid behavioural shifts. Their traditional MAPE was 18–20%.

CVS partnered with WorldQuant Predictive (WQP) to build a Data Proxy Model that replaced historical sales as the primary demand signal with five alternative data sources: Zillow real estate data (r=0.847), JHU/NYT infection rates, Department of Labor employment figures, GDELT news sentiment scores, and Descartes Lab mobility analytics — all anchored to ZIP-code level geographic areas.

The result: forecast accuracy improved to ±2% MAPE, enabling CVS to pre-position inventory, reduce stockouts, and maintain supply chain performance through unprecedented disruption. This system is the subject of this data model reference.
